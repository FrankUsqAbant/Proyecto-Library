# Leer es Pensar 📚

Una biblioteca digital moderna y elegante diseñada para los amantes de la lectura. Explora miles de libros clásicos, gestiona tu colección personal y disfruta de una experiencia de lectura premium.

## ✨ Características Principales

- **Biblioteca Inteligente**: Acceso a miles de títulos clásicos a través de la API de Gutendex.
- **Búsqueda Avanzada**: Búsqueda por título, autor y por voz integradas.
- **Colección Personal**: Guarda tus libros favoritos para leerlos más tarde.
- **Modo Lectura Premium**: Interfaz de lectura limpia con opciones de personalización (próximamente).
- **Diseño Responsive**: Optimizado para dispositivos móviles, tablets y escritorio.
- **Modo Oscuro**: Una experiencia visual relajante para lecturas nocturnas.
- **PWA Ready**: Instala la aplicación en tu dispositivo para un acceso rápido.

## 🚀 Tecnologías Utilizadas

- **Next.js 15+** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS** (v4)
- **Framer Motion** (Animaciones)
- **Lucide React** (Iconografía)

## 📦 Instalación y Uso

1. Instalar dependencias:

   ```bash
   npm install
   ```

2. Iniciar servidor de desarrollo:

   ```bash
   npm run dev
   ```

3. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.

## 🛠️ Estructura del Proyecto

- `app/`: Rutas y lógica de páginas.
- `components/`: Componentes de UI reutilizables y modulares.
- `hooks/`: Lógica de estado y efectos personalizada.
- `lib/`: Utilidades, constantes y cliente de API.
- `public/`: Activos estáticos y configuración de PWA.
