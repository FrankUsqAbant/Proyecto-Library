# 🎨 UI/UX Design System Skill

Esta skill define los estándares de interfaz y experiencia de usuario para el lector digital.

## 📌 Estándares Visuales

1. **Diseño Premium**:
   - Paletas armónicas en modo oscuro/claro con gradientes sutiles.
   - Efectos Glassmorphism y elevaciones mediante sombras responsivas.
   - Iconografía limpia usando `lucide-react`.

2. **Micro-interacciones**:
   - Transiciones fluidas con `framer-motion`.
   - Hover states bien definidos y feedback visual inmediato (toasts, skeletons).

3. **Accesibilidad (WCAG 2.1)**:
   - Contraste de color suficiente en textos.
   - Atributos ARIA en elementos interactivos modales y botones icono.
